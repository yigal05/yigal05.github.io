Graficadora Matematica

Desarrollé una graficadora matemática capaz de representar funciones básicas, diseñada para facilitar el aprendizaje visual de conceptos algebraicos y cálculo. La aplicación permite ingresar expresiones matemáticas, ajustar rangos y observar solo los cuadrandes donde la gráfica está presente.

DatAgranomy

Datagronomy es una aplicación de escritorio desarrollada en Python con Tkinter, diseñada para facilitar la gestión productiva y comercial en granjas porcinas. La herramienta centraliza procesos clave relacionados con el manejo de animales, inventario, clientes y ventas, incorporando roles de usuario y reportes básicos para una mejor toma de decisiones.

PixelDom (Tienda de Productos)

Desarrollé una aplicación de escritorio robusta y modular utilizando Java y Swing, diseñada para la gestión integral de usuarios y productos. Este sistema implementa tecnicas de clean code, además, tiene sistemas de identificación distinguiendo entre roles de administrador y usuarios regulares para acceder a funcionalidades específicas.
